# Features

## Hecho ✅
- **Separar login / crear cuenta** — login directo con usuario+contraseña; avisa que el usuario es inicial + apellido. Recuerda el último usuario.
- **Toggle de crear cuenta** — desde el panel admin (pestaña Funciones).
- **Panel admin "Funciones"** — toggles para prender/apagar features en vivo.
- **Countdown al próximo partido sin pronosticar** — banner con cuenta regresiva. (toggle: `countdown_enabled`)
- **Racha** — cuántos partidos seguidos sumaste puntos, columna en la tabla. (toggle: `streak_enabled`)
- **Podio visual top 3** en la home + posición y puntos del usuario como 4to. (toggle: `podium_enabled`)
- **Lock visual** en partidos cerrados — muestra el pronóstico en modo lectura ("Tu pronóstico: X-Y").
- **Rediseño de la card de partido** — banderas, badge centrado (fecha/puntos/vivo), info desplegable, alineación.
